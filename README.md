## 📋 Posting Guide
**파일명 규칙**: `YYYYMMDD-학습키워드.md` (예: `20260326-controller-service-separation.md`)

| 카테고리 | 세부 항목 | 작성 내용 |
| :--- | :--- | :--- |
| **정보기록** | 개념 요약 / 코드 예시 | 새로 배운 개념 정의와 핵심 코드 스니펫 |
| **구현과정** | 배경 / 기술 / 구조 | 기능 도입 이유, 스택, 데이터의 흐름(Flow) |
| **문제해결** | 원인 / 분석 / 해결 | 에러 로그 분석, 트러블슈팅 과정 및 배운 점 |
| **기술비교** | 주제 / 차이점 / 선택 | 기술 간 장단점 비교 및 최종 의사결정 이유 |
| **회고** | 자유 기록 | 시행착오, 느낀 점, 다음 미션 다짐 |
