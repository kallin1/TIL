## 시간 복잡도 구하는 방법

1. O 표기법만 구할 때
중첩 루프 k개가 각각 n에 비례하면 O(n^k) 로 끝.
계수나 상수는 전부 버림.

2. 정확한 수행 횟수를 구할 때
루프 인덱스가 서로 의존하는 구조면 조합(nCr) 으로 접근.
구조의미횟수j = i+1 (2중)n개 중 2개 선택n(n-1)/2k = j+1 (3중)n개 중 3개 선택n(n-1)(n-2)/6
수열의 합으로 풀어도 결과 동일.

3. 특수한 경우
인덱스가 배수로 증가 (i *= 2) → O(log n)
재귀가 두 번 호출 (피보나치) → O(2^n)


## 핵심 정리
O 표기법 → 최고차항만, 계수 버리기
정확한 횟수 → 조합 공식 사용
n = 500,000이면 n^2 = 2,500억 → Long 필수
